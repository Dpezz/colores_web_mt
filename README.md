## Instalar y correr el proyecto

```bash
$ git clone https://github.com/Dpezz/colores_web_mt.git
$ cd colores_web_mt/
$ npm install
$ ng serve  #or npm start

start server [http://localhost:4200](http://localhost:4200)
```
